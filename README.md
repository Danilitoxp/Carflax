## 🌟 Visão Geral

O **Carflax HUB** é uma plataforma corporativa de alto desempenho projetada para unificar as operações de vendas, logística e CRM da Carflax em uma única interface elegante e intuitiva. Construído com o **Midnight Design System**, o HUB oferece uma experiência imersiva com efeitos visuais de ponta e performance ultrápida.

## ✨ Destaques Visuais & Experience

- 🌌 **Sparkles Engine**: Efeito de partículas dinâmicas no cabeçalho que reage ao tema do sistema.
- 🌓 **Adaptive UI**: Design inteligente que alterna entre os modos Light e Dark com paletas de cores sintonizadas.
- 💎 **Glassmorphism**: Uso de transparências e borrões (backdrops) para uma sensação de profundidade e luxo.
- 🚀 **Performance First**: Interface otimizada com Vite e Componentes Shadcn para carregamento instantâneo.

## 🛠️ Módulos Principais

### 📊 CRM & Vendas
- **Inteligência de Perdas**: Ranking gamificado e análise granular de perdas por itens de estoque e preço.
- **Relatórios Automatizados**: Extração ágil de métricas de conversão e performance de vendedores.
- **Privacidade Role-Based**: Filtros automáticos que garantem que cada vendedor visualize apenas seus próprios dados.

### 🚚 Logística & Entregas
- **Gestão de Romaneios**: Controle absoluto sobre o fluxo de mercadorias e status de entregas.
- **Operational Dashboards**: Visão clara de entregas pendentes, concluídas e romaneios ativos.

### 📅 Calendário Corporativo
- Gestão centralizada de eventos, treinamentos e escala de férias da equipe em tempo real.

### 💬 Comunicação Realtime
- Chat global integrado e sistema de avisos dinâmicos alimentado por Supabase Realtime.

## 🚀 Tecnologias

- **Frontend**: React 18, Vite, Tailwind CSS
- **Componentes**: Lucide Icons, Radix UI, Shadcn/UI
- **Backend / Realtime**: Supabase (PostgreSQL)
- **Efeitos**: @tsparticles (Sparkles Engine)

## 💻 Desenvolvimento

### Pré-requisitos
- Node.js (v18+)
- NPM ou Yarn

### Instalação

1. Clone o repositório oficial da Carflax:
   ```bash
   git clone git@github.com:Carflax/Carflax.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Configure as variáveis de ambiente (`.env`):
   ```env
   VITE_SUPABASE_URL=seu_url_do_supabase
   VITE_SUPABASE_ANON_KEY=sua_chave_anonima
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
